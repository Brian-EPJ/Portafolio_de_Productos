Sobre mí

¿Quién soy?

Soy Brian Pradenas, un desarrollador Full Stack en formación apasionado por la tecnología y el desarrollo de software. Mi enfoque principal está en el desarrollo backend con Java, donde disfruto construyendo aplicaciones robustas, escalables y eficientes que resuelven problemas reales.

Experiencia y Formación

Cuento con experiencia práctica en el desarrollo de aplicaciones web utilizando el ecosistema Java, incluyendo Spring Boot para crear APIs REST y aplicaciones web con Thymeleaf, JPA/Hibernate para la persistencia de datos, y Jakarta EE para aplicaciones empresariales. He trabajado en proyectos que van desde sistemas de gestión de bibliotecas hasta sitios web dinámicos para bandas de música.

Mi formación incluye el manejo de bases de datos relacionales con MySQL, diseño de modelos de datos, y la implementación de arquitecturas MVC. Además, tengo conocimientos en tecnologías frontend como HTML, CSS y JavaScript para crear interfaces funcionales y atractivas. Utilizo IntelliJ IDEA como mi IDE principal y Git para control de versiones.

Objetivo Profesional

Mi objetivo es continuar creciendo como desarrollador Full Stack, especializándome en tecnologías Java y frameworks modernos. Busco oportunidades donde pueda aportar valor mediante la creación de soluciones innovadoras, trabajar en equipo con otros profesionales y seguir aprendiendo las mejores prácticas de la industria.

Aspiro a formar parte de proyectos desafiantes donde pueda aplicar mis conocimientos técnicos, contribuir al éxito del equipo y desarrollar software de calidad que impacte positivamente a los usuarios.
